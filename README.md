# Project1_Predictive-Maintenance
# Predictive Maintenance using Contextual Data Fusion

## Project Overview
This project aims to predict machine failures by combining industrial sensor telemetry data with external weather information.

## Week 1: IoT Telemetry Ingestion and Signal Processing

### Objectives
- Load and preprocess AI4I Predictive Maintenance dataset
- Integrate external weather data
- Perform telemetry signal processing
- Generate rolling statistical features

### Dataset Features
#### Machine Sensor Data
- Air Temperature
- Process Temperature
- Rotational Speed
- Torque
- Tool Wear

#### Weather Data
- Average Temperature
- Minimum Temperature
- Maximum Temperature
- Precipitation
- Wind Speed
- Sea Level Pressure

### Tasks Completed
- Data Loading using Pandas
- Data Cleaning and Column Standardization
- Time-Series Data Sorting
- Sensor and Weather Data Integration
- Rolling Mean Feature Engineering
- Rolling Standard Deviation Feature Engineering
- Rolling Variance Feature Engineering

### Technologies Used
- Python
- Pandas
- NumPy
- Jupyter Notebook

### Output
Generated baseline telemetry features for predictive maintenance modeling and future machine learning tasks.
